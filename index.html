Identifica y describe las propiedades de los siguientes elementos CSS:

Tema 1-Texto

Propiedad Valor Inicial Descripción
color black Define el color del texto.
font-family serif Especifica la fuente del texto.
font-size medium Establece el tamaño de la fuente.
font-weight normal Controla el grosor del texto (normal, bold, etc.).
font-style normal Define si el texto es normal, cursivo o itálico.
font-variant normal Especifica el estilo de la fuente (normal o pequeño cap).
font - Atajo para definir varias propiedades de fuente a la vez.
text-align start Alinea el texto (izquierda, derecha, centro, justificado).
line-height normal Controla la altura de la línea entre el texto.
text-decoration none Especifica la decoración del texto (subrayado, tachado, etc.).
text-transform none Controla la capitalización del texto (mayúsculas, minúsculas, etc.).
vertical-align baseline Alinea elementos en línea verticalmente (top, middle, bottom).
text-indent 0 Indenta el texto de la primera línea.
letter-spacing normal Controla el espaciado entre letras.
word-spacing normal Controla el espaciado entre palabras.
white-space normal Controla cómo se manejan los espacios en blanco (normal, nowrap, etc.).

Tema 2-Enlaces

Característica Propiedades Descripción
Tamaño, Color y Decoración color Define el color del texto del enlace. Puede ser un color nombrado (como red).
font-size Establece el tamaño de la fuente del texto del enlace. Se puede especificar en unidades como píxeles (px), puntos (pt), ems (em), rems (rem), etc.
text-decoration Controla la decoración del enlace. Los valores comunes son: subrayado, tachado, etc.
Decoración Personalizada border Define el borde alrededor del enlace. Puedes especificar el ancho, el estilo y el color del borde. Ejemplo: border: 1px solid black;
border-radius Establece esquinas redondeadas en el borde del enlace. Puedes especificar un valor en píxeles o en porcentajes. Ejemplo: border-radius: 5px;
padding Controla el espacio interior entre el contenido del enlace y su borde. Permite aumentar el área clicable. Ejemplo: padding: 10px;
box-shadow Añade una sombra alrededor del enlace, creando un efecto de profundidad. Puedes especificar desplazamiento, desenfoque y color. Ejemplo: box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
text-shadow Aplica una sombra al texto del enlace, haciendo que se destaque. Define desplazamiento, desenfoque y color. Ejemplo: text-shadow: 1px 1px 2px rgba(0, 0, 0, 0.7);

Tema 3-Imagenes

Característica Propiedades Descripción
Anchura y Altura width Establece el ancho de la imagen. Ejemplo: width: 100%;
height Establece la altura de la imagen. Ejemplo: height: auto;
Eliminar el Borde height Establece la altura de la imagen. Ejemplo: height: auto;.

Tema 4-Listas

Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador de lista (disc, circle, square, none).
list-style-position outside Controla la posición del marcador (inside o outside).
list-style-image none Permite usar una imagen como marcador.
list-style - Atajo para definir list-style-type, list-style-position y list-style-image a la vez.

Tema 5-Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse <table> separate Define si los bordes de las celdas se colapsan o se separan.
border-spacing <table> 0 Establece el espacio entre los bordes de las celdas.
empty-cells <table> show Controla la visibilidad de las celdas vacías (show o hide).
caption-side <table> top Especifica la posición de la leyenda de la tabla (top, bottom, left, right).
